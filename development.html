Development

Dit is een vervolg op project 2 “Fix that UI”. In project 3 moeten we de website die we hiervoor hebben ontworpen, de “Zekur” verzekering aansluiting proces, programmeren. Hiervoor is ons groep in twee gesplits. Ik heb samen met Sara en Jonas gewerkt. Vorige project had Jonas alleen gewerkt waardoor hij andere prototypes heeft gemaakt. Wij hebben als groepje gekeken naar beide prototypes. We hebben de beste elementen uit beide gepakt en deze samengevoegd. Dit is het uiteindelijke flow prototype geworden.

Wij hebben aan het begin van het project een planning gemaakt voor ons zelf en zo hebben we wekelijks 1 tot 3 vergaderingen gehad. Elke keer hebben we taken verdeeld en de volgende keer deze laten zien. Jonas heeft mij uitgelegd hoe ik code moet pushen en pullen op Git. Op het begin snapte ik dit nog niet helemaal, maar ik heb op school ook hulp gevraagd aan andere studenten. Nu weet ik precies wat ik allemaal moet doen om mijn code te delen.

Programmeren

In de eerste week hebben we bestolen dat ik de inhoud van de landings pagina ga maken en Jonas de header en footer. Ik had op het begin moeite met het positioneren van een aantal elementen. Natuurlijk heb ik wel hulp gekregen van ChatGpt, maar ik heb ook veel dingen zelf geprobeerd. Nadat ik een stuk code had en het er ongeveer uitzag hoe ik het wilde hebben, heb ik veel nagechekt met “inspecteren” zo heb ik de positie van de teksten en afbeeldingen verfijnd.

Belangerijkste wat ik van deze samenwerking heb geleerd is dat het heel handig is om verschillende css bestanden te maken. Eerst zette ik alles in één bestand waardoor het heel erg onoverzichtelijk werd. Ik heb bij mijn teamgenoot gezien dat hij aparte CSS bestanden heeft gemaakt voor dingen zoals: header, footer, kleuren, cards, enz. Dit zorg er voor dat alles georganiseerd blijft. Dit heb ik dus ook meteen toegepast bij mijn eigen portfolio website.

Omdat Jonas stap 1 heeft gemaakt, had hij al code geschreven voor de kaarten. Deze heb ik kunnen kopieren en plakken in mijn eigen bestand. Ik heb een aantal dingen aangepast en zo heb ik makkelijk mijn pagina's kunnen maken.

We wisten al dat wij veel Javascript nodig gaan hebben bij het maken van ons website. Eerst wilde ik dit allemaal overlaten en mijn collega. Maar toen dacht ik er aan dat ik deze kennis ook nodig heb. Ik heb dus onderzoek gedaan en kwam er achter dat het niet eens zo heel moeilijk is. Bij stap 3 en 4 hebben we een dropdown nodig. Deze heb ik dus gemaakt bij stap 4, wat mijn pagina was. Ik heb ook een knop geprobeerd toe te voegen wat de eerste section zou dichtklappen en de twee uitklappen. Maar dit was niet meteen gelukt dus dat heb ik nu nog niet. Ik me hier wel nog meer in verdiepen tot het werkt.
