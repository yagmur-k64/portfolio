Dream Organisation

Voor het laatste project van dit semester mogen wij zelf kiezen wat we willen doen. Ik heb een bedrijf gevonden wat nog wat hulp kan gebruiken. Ik denk dat ik tijdens het maken van dit project alle leerdoelen kan tonen. Voor development omdat ik bij de vorige projecten nog niet goed heb kunnen laten zien. Ik heb nu geleerd hoe ik moet werken met Git dus nu kan ik version controll doen. Ook ben ik van plan om meer te testen tijdens dit project.

voorstel

Dream Organisation is een evenementenorganisatiebedrijf gevestigd in Helmond. Ze verzorgen alles, van decoratie tot catering. De eigenaar van Dream Organisation is Emine Gokmen, een 42-jarige vrouw met een passie voor organiseren. Ze is al erg bekend in haar branche en heeft een groot publiek opgebouwd. Toch verliest ze nog steeds veel klanten omdat ze weinig aandacht besteedt aan branding.

Een korte tijd geleden is de Dream Organisation team begonnen met het maken van online content op platforms zoals TikTok en Instagram. Dit heeft haar al meer volgers opgeleverd. Maar er ontbreekt nog steeds een cruciaal middel: een professionele website. Dream Organisation heeft momenteel geen eigen website.

Ik zou graag een website willen maken voor Dream Organisation. Hierin zou ik afbeeldingen van verschillende decoratiepakketten toevoegen, zodat klanten deze als een catalogus kunnen bekijken. Daarnaast zou de website een functie bevatten waarmee bezoekers eenvoudig een bericht kunnen sturen om een offerte aan te vragen.

Momenteel wordt Emine vaak gebeld door klanten met vragen. Daarom stel ik voor om een chatfunctie op de website te integreren, zodat klanten eenvoudige vragen direct via de website kunnen stellen. Dit bespaart tijd en maakt de communicatie efficiënter.

Verder beschikt Dream Organisation over een grote witte bus zonder enige reclame. Dit is natuurlijk een gemiste kans. Emine en haar team reizen regelmatig door heel Nederland, en een bestickerde bus zou uitstekende rijdende reclame zijn. Ook wanneer de bus geparkeerd staat voor huizen tijdens feestjes, biedt dit de mogelijkheid om de naamsbekendheid te vergroten. Op zulke momenten zien veel gasten, en potentiële klanten, de bus.

Met deze ideeën, en meer, kan ik helpen om Dream Organisation nog meer bekendheid te geven en verder te laten groeien.
